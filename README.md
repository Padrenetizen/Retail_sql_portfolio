Retail Data Cleaning & Analysis (MySQL)

## Overview
This project involves cleaning, validating, and analyzing retail sales data using MySQL.

## Tools Used
- MySQL
- SQL
- GitHub

## Dataset
Retail transactions including invoice details, products, quantity, price, and country.

## Data Cleaning Steps
- Removed duplicates and invalid records
- Handled missing values
- Standardized date formats
- Created reusable SQL views

## SQL Views
- view1: cleaned dataset
- sales_view: line-level revenue calculations

## Key Insights
- The UK, Netherlands, & Ireland generated the highest revenues and order volume
- Sales peak in Q4, showing strong seasonal demand
- A few products contribute to most of the revenue
